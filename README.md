# progAlea
Program assign a random person

## To run with docker

Install [Docker](https://docs.docker.com/engine/installation/)

Open a bash terminal (Git bash is fine for Windows) and type:

`./build.bash`

`./run.bash`

Go to [localhost:80](http://localhost:80)
